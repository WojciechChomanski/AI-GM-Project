# Asha’s Chain-Song (Fragment)

Barukh ata… adonai… Thanatos ou dynatai kataschesai…  
Mavet lo yichyeh, lo yichyeh… In umbra mea, ego sum… ego sum…  
…Barukh… adonai… elohai…  
(Chains strike stone)
